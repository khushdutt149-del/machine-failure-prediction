# Machine Failure Prediction System

## Project Overview

This project predicts machine failures using Machine Learning based on sensor readings collected from industrial equipment. The application helps identify potential failures before they occur, enabling predictive maintenance and reducing downtime.

## Features

* Real-time machine failure prediction
* Interactive Streamlit web application
* Random Forest classification model
* Sensor-based predictive analytics
* Easy-to-use dashboard

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Random Forest

## Dataset Features

* Footfall
* Temp Mode
* AQ (Air Quality)
* USS (Ultrasonic Sensor)
* CS (Current Sensor)
* VOC (Volatile Organic Compounds)
* RP (Rotational Position)
* IP (Input Pressure)
* Temperature

## Model Performance

* Logistic Regression Accuracy: 91.0%
* Random Forest Accuracy: 92.6%

## Deployment

The application is deployed using Streamlit Community Cloud.
